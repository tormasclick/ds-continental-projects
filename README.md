# DS Continental Projects - Corporate Website

A multi-page corporate website for DS Continental Projects, showcasing its industrial group and subsidiaries including:
- DS Continental Motors (Heavy equipment & machinery)
- DS Continental Security (Security solutions)  
- DS Continental Trading (Mineral trading & mining)

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn/ui

## Development

\`\`\`bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
\`\`\`
